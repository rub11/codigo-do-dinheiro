Box Model e Unidades CSS

Domine o controle de tamanhos, espaçamentos e layout

1 Tamanhos, Margens e Espaçamento

Aprenda a ajustar tamanho, margem e espaçamento dos elementos. Controle de width, height, margin, padding e suas variações.

width: 300px;
height: 200px;
margin: 20px;
padding: 15px;

2 Unidades de Medida

Domine as principais unidades CSS: px (absoluta), em e rem (relativas), % (porcentagem), vw/vh (viewport) e quando usar cada uma.

font-size: 16px; /* Absoluto */
font-size: 1.5rem; /* Relativo ao root */
width: 50%; /* Porcentagem */
height: 100vh; /* Viewport */
Unidades PX REM/EM

4 Margin, Padding e Border

Domine as propriedades de espaçamento: margin (externo), padding (interno) e border (borda). Shorthand properties e direções específicas.

margin: 10px 20px; /* top/bottom left/right */
padding: 15px 10px 5px; /* top left/right bottom */
border: 2px solid red; /* width style color */
Margin Padding Border

5 Box-Sizing

Entenda a propriedade box-sizing: content-box vs border-box. Como controlar se padding e border são incluídos no width/height.

box-sizing: content-box; /* padrão */
width: 300px + padding + border

box-sizing: border-box; /* moderno */
width: 300px (inclui padding e border)
Box-Sizing Layout Controle

6 Prática: Layout Responsivo

Aplicação prática de todos os conceitos em um layout responsivo usando Box Model, unidades relativas e box-sizing border-box.

/* Reset moderno */
* { box-sizing: border-box; }

/* Layout responsivo */
.container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 2rem;
}
